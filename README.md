
# Paystar - Task
## Run Locally

Clone the project

```bash
  git clone https://github.com/federicoalbanese/accounting-task
```

Go to the project directory

```bash
  cd accounting-task
```

Install dependencies

```bash
  composer install
```

Start the server

```bash
  php artisan serv
```

## Postman Collection 
##### in the root of project file name : accounting-task.postman_collection.json
